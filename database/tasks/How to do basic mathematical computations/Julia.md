| Mathematical notation | Julia code |
|--|--|
| $x+y$ | `x+y` |
| $x-y$ | `x-y` |
| $xy$ | `x*y` |
| $\frac xy$ | `x/y` (or `y\x`) |
| $\left\lfloor\frac xy\right\rfloor$ | `x√∑y` |
| remainder of $x\div y$ | `x%y` |
| $x^y$ | `x^y` |
| $\vert x\vert$ | `abs(x)` |
| $\ln x$ | `log(x)` |
| $\log_a b$ | `log(a,b)` |
| $e^x$ | `exp(x)` |
| $\pi$ | `pi` |
| $\sin x$ | `sin(x)` |
| $\sin^{-1} x$ | `asin(x)` |
| $\sqrt x$ | `sqrt(x)` |

Other trigonometric functions are also available besides just `sin`
including `cos`, `tan`, etc.
