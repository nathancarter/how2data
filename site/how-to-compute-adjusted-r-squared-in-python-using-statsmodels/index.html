How to compute adjusted R-squared (in Python, using statsmodels)

See all solutions.

Task

If we have fit a multiple linear regression model, how can we compute the Adjusted $R^2$ for that model, to measure its goodness of fit?

Related tasks:

  • How to compute R-squared for a simple linear model

Solution

We assume you have already fit a multiple linear regression model to some data, as in the code below. (If youâ€™re unfamiliar with how to do so, see how to fit a multiple linear regression model.) The data shown below is fake, and we assume you will replace it with your own real data if you use this code.

import pandas as pd
import statsmodels.api as sm
df = pd.DataFrame( {
    'x1':[2, 7, 4, 3, 11, 18, 6, 15, 9, 12],
    'x2':[4, 6, 10, 1, 18, 11, 8, 20, 16, 13],
    'x3':[11, 16, 20, 6, 14, 8, 5, 23, 13, 10],
    'y':[24, 60, 32, 29, 90, 45, 130, 76, 100, 120]
} )
xs = df[['x1', 'x2', 'x3']]
y = df['y']
xs = sm.add_constant(xs)
model = sm.OLS(y, xs).fit()
